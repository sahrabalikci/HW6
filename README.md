# To do List App

A to do list application built with HTML, CSS, and JavaScript.
This project was built with the help of the Cursor AI vibe coding tool.

## Features

- Add tasks with title and due date
- Edit and delete tasks
- Mark tasks as complete (automatically moves to Completed section)
- LocalStorage persistence (tasks survive page refresh)
- Responsive design
